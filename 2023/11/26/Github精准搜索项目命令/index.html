Github精准搜索项目命令

1.in: 根据某个关键词来进行检索

关键词:
name: 项目名称
description: 项目描还
readme: 项目帮助文档
语法:

需要检索的内容 in:name或description或readme

单独使用:
(1) 列如: 检余项目名称包含vue的项目

vue in:name

(2)列如: 检索项目描述中包含vue的项目

vue in:description

(3)列如: 检索项自帮助文档中包含vue的项目

vue in:readme

组合使用:

(1) 列如:

检索项目名、项目描述、项目帮助文档中包含vue的项目
vue in:name,description,readme

2、根据stars或fork关键词查找单独使用语法:

单独使用语法:

(1) 数量范围: xxx 关键词 stars 通配符 :>= 或者 :<=

列如: 查找stars数量大于等于2000的vue项目

vue stars:>=2000

(2) 区间范围: xxx 关键词 fork: 数字1…数字2

列如: 查找fork 数量500到2000的springboot项目

springboot fork:500…2000

组合使用语法: xxx 关键词 stars 数字1…数字2 fork 数字1…数字2 从数字1到数字2

列如: 查找redis stars数 从500到2000 fork数从200到2000的项目

redis stars:500…2000 fork:200…2000

3、高亮显示代码#L

(1)高亮显示某一行

语法: 项目URL#L要显示的行号

列如: 高亮显示第20行代码

项目URL#L20

(2)高亮显示某一段代码

语法: 项目URL#L代码开头行号-L结束行号

列如: 高亮显示第20行到第40行代码

项目URL#L20-L40

4、搜索学习资料 awesome

语法: awesome 关键字

列如: 查找vue学习资料

awesome vue stars:>=2000

5、同性交友 location和language

语法: location:地区language:语言

列如:查找成都地区的java语言大佬

location:chengdu language;java
