文件上传（FTP）的使用：

第一步：安装vsftpd


注：yum install vsftpd(yum -y install vsftpd)

第二步：打开上传和写入配置：


vi /etc/vsftpd/vsftpd.conf
anon_upload_enable=YES
anon_mkdir_write_enable=YES

第三步：开启服务


systemctl enable vsftpd.service
systemctl start vsftpd.service
systemctl status vsftpd.service

第四步：关闭防火墙（远程连接时需要）


systemctl stop firewalld.service
systemctl disable firewalld.service

第五步：打开文件读写权限


1)、开放权限：chmod 777 -R /var/ftp/pub
2)、进入配置文件：vi /etc/selinux/config 将SELINUX设成disabled
3)、重启机器:reboot

安装MySQL数据库

1、安装MySQL数据库


sudo rpm -ivh mysql-community-release-el7-5.noarch.rpm
sudo yum install mysql-server

2、修改密码


1)、给目录授权：chown root /var/lib/mysql
2)、修改MySQL的登录设置：vi /etc/my.cnf（在[mysqld]的段后加上一句：skip-grant-tables）
3)、启动MySQL服务：service mysqld start

1)、mysql
2)、选择数据库：use mysql;
3)、update user set Password=password(‘123456’) where User=’root’;
4)、quit
5)、删除/etc/my.cnf里的[mysqld]段中 加上：skip-granat-tables,并尝试登录：mysql:mysql -u root -p
6)、启动MySQL服务：source /etc/profile

3、开放远程连接权限：


1、开放3306端口：/sbin/iptables -I INPUT -p tcp –dport 3306 -j ACCEPT
