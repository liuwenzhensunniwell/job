Termius 破解

发表于2023-11-26|更新于2023-11-26|工具工具破解
|阅读量:

一、windows破解

  • 版本说明：下载 termius -Version 8.1.2 到 -Version 7.5.8 都可以

1. 下载地址：https://termius.com/

2. 直接安装

进来发现没有sftp上传文件，说明需要付费才能sftp上传文件，那么进行第3步破解。

3. 破解termius

3.1. 安装node.js/npm环境

先打开cmd，检查下有没有npm环境

**
** 如果没有请去安装npm和node。官网地址：http://nodejs.cn/download/，建议下载14任意版本，比较稳定，安装过的这步骤不需要去管

3.2. 执行命令

打开cmd，安装asar，用来解压配置文件

输入：npm install -g asar

3.3. 进入termius配置目录

  • 找到自己电脑的 termius 安装目录 目录栏输入 cmd 进入

是你的 termius 目录，如果你找不到，可以右键 termius 软件，打开文件所在位置即可看到了。

  • 输入 asar e app.asar app 回车

  • 注意：若执行 命令 报 npm WARN deprecated asar@3.2.0: Please use @electron/asar moving forward. There is no API change, just a package name change，说明安装时没有对asar添加环境变量，需要自己找到asar安装目录，添加环境变量。

  • 解决方法：

  1. 打开cmd，执行命令：npm config get prefix，可以看到npm安装的命令路径

  1. 复制安装路径，接下来配置环境变量

**
**打开控制面板，搜索“高级系统”，点击查看高级系统设置

  1. 点击Path，选择编辑

  1. 选择新建，粘贴刚刚复制的安装路径，点击确定，外层再点击一次确定，保存编辑

  1. 重新打开cmd，输入全局安装的命令，可以看到，已经能访问了

3.4. 修改文件

  • 1.删除app-update.yml文件，因为需要断网破解，不然有可能不成功.

  • 2.使用vscode打开ui-process.js文件，vscode打开目录 app\js\ui-process.js。ctrl+a全选，然后右键，格式化代码一下

  • 搜索 return null === (t = e.user.team) || void 0 === t ? void 0 : t.owner ，修改成图片红色框的样子

4. 测试

  • 记得关了然后重新打开termius，使用sftp上传文件

  • 有sftp说明破解成功！！！！
